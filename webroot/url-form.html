Boston Police Journal Parser

Enter the URL to a BPD journal in PDF format for parsing.

Try https://boston-police-journal-parser.herokuapp.com/tests/pdfs/district_journal_arrests_public.pdf if you don't have a journal handy.
Please enter a valid URL to a PDF file.

This service is tightly coupled to the specific PDF format in which the BPD provides the journal. It will not work with other PDFs or other files.
See an Example Boston Police Department Public Journal .
